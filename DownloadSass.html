Sass Implementation

This is a website that uses a CSS Preprocessor to create the style.

A CSS Preprocessor can be used to organize CSS files and to minimize repetitive code. It also help to make the CSS file easier to read.

This was created using the Sass Preprocessor. Sass is an older version language that has its own syntax. They have created a newer version called SCSS that works with both Sass and CSS.
