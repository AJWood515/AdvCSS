Research and Define CSS Terms

Specificity- a ranking on css properties that will be applied to an element.

Precedence- a way to deal with mutiple style changes to a single element. Depends on the Specificity.

Inheritance- a child element would gain the properties of a parent element.

Property- define how the styles will look and where it will be placed on a wepage.

Value- a color, size, and other things given to a property to make changes to the property.

Selector- selects the content you wish to style.

Selector and Selector Specificity

Element- selects all elements with that specified element name. This is best used when specificity on the named element is not an issue such as want to change all the text in paragraphs a certain color or have the same hover effect for all links. The symbol for this is the element tag followed by curly brackets.

Group- groups element selectors together. This can be best used when you want a group of elements to have the same property. This is used by listing the elements and seperating them by a comma.

Descendent- a selector that will match all child or decendents of that specfic element. This is best used when you are trying to style an element within another element. this is used by listing the parent element first followed by a space then the child element you wish to style.

Class- selects elements with a specific class attribute. This is best used when you want specify the exact element you wish to use. This is used by using the period then a name of your choosing for the class.

Id- selector that uses an id to style a certain element with that id. It is best used when you want to style a certain element with a specific id. The symbol used for an id selector is the pound sign (#) followed by the name of the id.

Universal- this selector selects all of the elements. The best way to use this selector is to not really use it at all because this will select every thing but to use it as descendent selectorin certian situations might be okay. The symbol for this is the asterick (*).

Pseudo- selecter that selects an elemenet's special state and style such as hover, visted, active, and others. This is best used when you wish to style a element that can change states. This selector uses the colon symbol (:) followed by the pseudo-class.

The order of specificity for these selectors from least specific to most is unviersal, pseudo, element, group, descendent, class, and id.
